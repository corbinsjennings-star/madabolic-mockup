# MADabolic Homepage Mockup

Static homepage mockup for MADabolic.

## Files

- `index.html` — the full homepage
- `package.json` — Node config so Railway can serve it as a static site
- `.gitignore` — ignores `node_modules` and local junk

## Local preview

Open `index.html` in a browser. That's it.

## Deploy

Either GitHub Pages (free, instant) or Railway (needs the package.json above).
